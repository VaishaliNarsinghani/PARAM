# PARAM
Param is a role-based property-valuation workflow platform for end-to-end ops: Initiate → Assign → Visit → Draft → Review → Deliver. Six profiles (Initiator, Coordinator, FE, Report Drafter, Technical Manager, CSO), GPS-aware Android FE, path-based images (≤50), live maps, SLA flags, audit trails, and fast MySQL/LAMP deploy.
